# Chrome Web Store Screenshot Guide

Chrome Web Store requires **REAL** screenshots of your extension in action. AI-generated images cannot replace these as they need to show the actual UI.

## Required Screenshots (1280x800 recommended)

You need to upload at least 1 (recommended 3-5) screenshots.

### 1. The "Detected" State (Main Feature)
*   **Action:** Go to a site with a video (e.g., Vimeo, DailyMotion, or a news site).
*   **Setup:** Ensure the extension badge shows a number (e.g., "3").
*   **Capture:** Take a full-screen screenshot showing the browser, the video on the page, and the extension icon with the badge.
*   **Why:** Proves the detection works.

### 2. The Popup UI (Core Interface)
*   **Action:** Click the extension icon to open the popup.
*   **Setup:** Ensure the popup lists detected videos (MP4, HLS/M3U8).
*   **Capture:** Take a screenshot focusing on the popup open over the webpage.
*   **Tip:** You can use "Window Capture" to get just the browser window.

### 3. HLS Download in Progress (Advanced Feature)
*   **Action:** Start an HLS (m3u8) download.
*   **Setup:** Wait for the progress bar to appear (e.g., "Downloading... 45%").
*   **Capture:** Screenshot this state.
*   **Why:** Differentiates your extension from simple link grabbers. Shows the "Pro" processing capability.

### 4. Completed State
*   **Action:** Finish a download.
*   **Setup:** Show the "Download Complete" message or the file saved in the browser's download bar.
*   **Capture:** Screenshot.

## Technical Requirements
*   **Format:** JPEG or PNG (PNG recommended for clarity).
*   **Size:** 1280x800 or 640x400 pixels.
*   **Constraints:** No sensitive user data (blur out emails or private info if visible).

## How to take high-quality screenshots on Mac
1.  Press `Cmd + Shift + 4`, then press `Space`.
2.  Click on the Chrome window.
3.  This captures the window with a nice shadow and transparent background (ideal for marketing).
