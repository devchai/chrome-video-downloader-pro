# AI Image Generation Prompts (Nano Banana Pro)

Use these prompts with Midjourney, DALL-E 3, or similar AI image generation tools to create your store assets.

## 1. Store Icon (128x128)
*   **Purpose:** The main logo appearing in the browser toolbar and store listing.
*   **Design Goal:** Clean, recognizable at small sizes, "Pro" aesthetic.

**Prompt:**
```text
/imagine prompt: app icon design, minimalist flat vector graphic, white downward arrow seamlessly merging with a video play button symbol, set against a vibrant deep blue to electric purple gradient background, rounded square shape, modern material design, high contrast, clean lines, professional UI element, 8k resolution --no text, --no realistic shadows, --no complex details, --no photorealism
```

## 2. Marquee Promo Tile (1400x560)
*   **Purpose:** The large banner at the top of the Chrome Web Store listing.
*   **Design Goal:** High-tech, trustworthy, visualizing "streaming to file".

**Prompt:**
```text
/imagine prompt: software store marquee banner, abstract digital landscape, visualization of streaming video data flowing and coalescing into a solid secure file block, glowing neon blue and cyan data streams, dark mode background with glassmorphism effects, 3d render, octane render, 8k resolution, professional software store aesthetic, clean and modern, high tech atmosphere, wide angle --ar 5:2 --no text, --no human figures, --no messy details, --no low resolution
```

## 3. Promotional Concept Art (Optional)
*   **Purpose:** Additional marketing material or social media posts.

**Prompt:**
```text
/imagine prompt: 3d isometric illustration of a web browser window extracting a video file from a web page, clean 3d style, blender render, clay render style, pastel blue and white color palette, soft lighting, cute and modern technology concept, high quality, 4k --ar 16:9 --no text
```
